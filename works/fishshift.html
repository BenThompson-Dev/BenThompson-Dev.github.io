FISH SHIFT

Gameplay Programming, Game, Technical, Level & UX Design

As part of the first semester of the Advanced 3D Games Engines and Scripting module, I created a fish puzzle game using Unreal Engine 4's blueprint system. In the project, each fish has it's own ability and the player needs to swap between them to get through the challenges in the level.

I worked on this project on my own, using royalty-free assets and designing the gameplay and technical aspect, as well as doing all the blueprint scripting, including implementing a system to easily manage character abilities. I also put time into polishing the experience, focusing on tweaking the design and adding features/UI elements to improve the user experience.

The game prototype is available to play now on itch.io.
